## Attributes

| 参数      | 说明               | 类型    | 可选值 | 默认值 |
| --------- | ------------------ | ------- | ------ | ------ |
| header    | 表头信息           | string  | ——     | ——     |
| footer    | 页脚信息           | string  | ——     | ——     |
| content   | 主体内容           | array   | ——     | []     |
| keys      | 循环对象键名       | string  | ——     | ——     |
| zebra     | 是否显示斑马纹     | boolean | ——     | ——     |
| num       | 是否显示序号       | boolean | ——     | false  |
| hoverShow | hover 状态显示阴影 | boolean | ——     | false  |

## Slots

| 插槽名称 | 说明说明       |
| -------- | -------------- |
| header   | 自定义头部内容 |
| footer   | 自定义底部内容 |
