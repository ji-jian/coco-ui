## Attributes

| 参数     | 说明           | 类型    | 可选值                        | 默认值 |
| -------- | -------------- | ------- | ----------------------------- | ------ |
| v-mod    | 绑定的值       | string  | ——                            | ——     |
| label    | Radio 的 value | string  | ——                            | ——     |
| name     | 原生 name 属性 | string  | ——                            | ——     |
| disabled | 是否禁用       | boolean | ——                            | false  |
| Boole    | 是否显示边框   | boolean | ——                            | false  |
| size     | 大小尺寸       | string  | large / medium / small / mini | large  |
